前端基础–学习笔记(包含前端视频)

Lesson1 前端基础

NO1.1 HTML

HTML说白了，就是我们常见的网页，这一个个网页就是由HTML语言书写构成。

  • HTML
    • 定义：是指超文本标记语言。超文本是指页面内可以包含图片、链接等非文字元素，HTML用于编写网页，且HTML文件的格式后缀都以.html或者.htm结尾；
    • 组成：HTML文件只有一个根标签<html></html>，其下由头标签<head></head>和体标签<body></body>两部分组成；
    • 基本语法格式：
      • HTML标签的书写格式：属性名="属性值"。
    • 常用标签：
      • <title>xxx</title>：HTML网页的页面标题标签，即打开HTML页面后，你所看到的网页的标题，一个HTML只能有一个该标签；
      • <hn></hn>：普通标题标签，n取值：1到6，字体逐渐变小，且自动换行；
      • <hr />：水平线标签，即在HTML页面中创建一条水平分割线；
        • size属性：水平线的大小，单位是像素px；
        • noshade属性：没有阴影效果，取值：noshade，表示显示纯色，不设置该属性，则显示一个阴影；
      • <font></font>：字体标签；
        • color属性：设置字体颜色；
        • size属性：设置字体大小，值越大，字体越大，取值1到7；
      • <b></b>：粗体标签；即字体变粗；
      • <i></i>：斜体标签；即字体倾斜；
      • <p></p>：段落标签，前后换行隔开；
      • <br></br>：换行标签；即插入单个换行；
      • <img />：图片标签；
        • src属性：即图片的加载路径；
        • width属性：图片宽度像素值或百分比；
        • height属性：图片高度像素值或百分比；
        • alt属性：图片提示信息，即当图片加载不出来，放在图片旁边的说明；
        • title属性：当鼠标放到照片上，会显示出来的信息；
      • <ul></ul>：无序列表标签；必须和
      • 配合使用才有意义；
        • type属性：用于设置列表前面的标志，即实心小圆圈(disc)，空心小圆圈(circle)，正方形小圆圈(square)；
      • <ol></ol>：有序列表标签；必须和
      • 配合使用才有意义；
        • start属性：表示从哪个顺序开始，如start=4，则从4开始，如start=5，从5开始；
        • reversed属性：倒序标记，即从start设置的数字，倒着标记，例如4321标记；
        • type属性：设置顺序的数字，1，a，A等等，按着顺序标记；
      • <a></a>：超链接标签；
        • href属性：表示要跳转的地址；
        • target属性：跳转到新地址时，根据取值决定是否打开一个新的页面；
      • <table></table>：表格标签，必须和行标签<tr></tr>、列标签<td></td>配合使用，还可以配合一种单元格，即<th></th>使用，这个标签叫做表头标签；
        • border属性：设置表格边框的像素，即边框的粗细；
        • width属性：表格宽度，单位为像素值或百分比；
        • height属性：表格高度，单位为像素值或百分比；
        • align属性：设置表格位置，即居中，靠左，靠右；
        • bgcolor属性：设置表格颜色；
        • cellspacing属性：设置表格中每个单元格之间的距离，单位为像素px；
        • cellpadding属性：设置表格中每个单元格与其内容之间的距离，单位为像素px；
      • <tr></tr>：行标签；
        • rowspan属性：跨行合并，rowspan”2”，合并2个单元格，再将对应的被合并的单元格删除即可；
        • align属性：设置单元格里的内容位置，即居中，靠左，靠右；
      • <td></td>：列标签；
        • colspan属性：跨列合并，colspan=”2”，合并2个单元格，再将对应的被合并的单元格删除即可；
        • align属性：设置单元格里的内容位置，即居中，靠左，靠右；
      • <th></th>：表头标签，内容默认加粗，居中；
      • <form></form>：表单标签，主要用来收集用户数据，所有需要提交到服务器端的表单项必须用表单标签括起来；
        • action属性：整个表单提交的位置，即将表单提交给某处；
        • method属性：表单提交的方式(方式有7种，常见的是get，post)；
        • form表单标签中常用的子标签：
          • <input />：输入框，一般用于用户输入数据；
            • type属性(10种取值，9种值常用)：
              • text：文本输入框，value设置默认值；
              • password：密码输入框，以*号代替文本，value设置默认值；
              • radio：单选框，checked设置默认值；
              • checkbox：复选框，checked设置默认值；
              • button：普通按钮；
              • reset：重置按钮，清空表单；
              • submit：提交按钮，提交表单数据给服务端；
              • image：图片按钮，按钮是一张图片(不常用)；
              • hidden：隐藏域，数据会提交，但是页面上看不到；
              • file：文件上传；
            • name属性：
              • 用于给标签起名字；
              • 还可以给单选框或者复选框分组(这两个必须要分组，name相同的为一组)；
              • 最重要的是，表单提交时，有name属性的数据才会被提交；
            • value属性：
              • 给按钮类的属性值文字赋值，即按钮上显示文字；
              • 用于表单提交时，单选框和复选框填写的内容被服务端准确的获取；
              • 用于表单提交时，下拉框选择的内容被服务端准确的获取；
        • <select></select>：下拉框，一般用于用户选择某个选项；
          • <option></option>：下拉框的内容，即下拉框的子标签，两者需要配合使用才有意义；
            • selected属性：设置默认值；
        • <textarea></textarea>：文本域，一般用于用户输入大量的文字信息，例如自我介绍；
          • rows属性：设置行数；
          • cols属性：设置列数；
      • <div></div>：div标签，是一个块级元素，特点是自动换行，常作为一个容器使用；
      • <span></span>：span标签，是一个行内元素，特点是不换行；

 

 

NO.1.2 CSS

CSS，你可以理解为静态渲染HTML页面。

  • CSS：

    • 定义：是指层叠样式表，主要用于渲染HTML中的文本内容、图片、以及版面布局等外观显示样式；

    • 基本语法格式：

      选择器{
	属性名:属性值;
	属性名:属性值;
}

    • 在HTML中引入CSS代码：

      • 行内样式：在标签的style属性中写CSS代码，此时只需要写： 属性名:属性值; 属性名:属性值; 即可，但是只对该标签有效；
      • 内部样式：在head标签的子标签style标签中写CSS代码，按照CSS正常格式写即可；
      • 外部样式：将CSS代码写在单独的文件中，后缀名以.css结尾，在head标签中通过link标签引入.css文件即可；

    • 选择器：用于选中某个或者某些标签，以便于我们给这些标签设置CSS样式；

      • 标签选择器：选择指定的标签；

        标签名{
	属性名:属性值;
}

      • id选择器：选择拥有指定id值的标签；(注意：该方式必须为标签添加一个id属性)

        #id值{
	属性名:属性值;
}

      • class选择器：选择拥有指定class值的标签；

        .class值{
	属性名:属性值;
}

      • 属性选择器：即选择拥有指定属性的标签；

        标签名[属性名:属性值]{
	属性名:属性值;
}

      • 后代选择器：即父标签包含后代标签；

        选择器1 选择器2{
	属性名:属性值;
}

    • CSS中常见的属性名：

      • border属性：边框样式，有三个值，值之间用空格隔开；如，border:1px solid red；
      • width属性：宽度；
      • height属性：高度；
      • color属性：内容的颜色；
      • font-size：内容文字的大小
      • .background-color：背景颜色；
      • display：转换样式，即块级元素和行内元素的互相转换，以及隐藏所选标签元素；
      • float：设置div浮动效果，默认向上飘后，向左飘，知道碰到其他标签；
      • clear：清除元素两边的浮动效果，值只选择both；

    • CSS盒子模型：CSS将每一个标签都认为是一个盒子，其拥有两个边距：

      • 内边距：即元素和边框的距离，属性名padding，默认元素与边框是重合的，即padding=0px，内边距越大，边框越大；
      • 外边距：即边框和其他边框的距离，属性名margin。

 

 

NO.1.3 JavaScript

JS，你可以理解为动态渲染HTML页面。

  • JavaScript(以下简称Js)：

    • 定义：是一种嵌入式脚本编程语言，用于开发交互式的web页面，使用JavaScript可以添加动画效果，提升用户操作体验；

    • 应用：

      • 嵌入动态文本于HTML页面中；
      • 对浏览器事件做出相应；
      • 读写HTML元素；
      • 验证提交数据；
      • 检测访客的浏览器信息等；

    • 组成：

      • ECMAScript：Js的核心语法；
      • BOM：浏览器对象；
      • DOM：操作文档中的元素和内容；

    • 引入方式：

      • 内嵌式：

        • 必须在<script></script>标签中写入Js代码；

        • 格式为：

          <script type="text/javascript">
	js代码
</script>

      • 外联式：

        • 在后缀名为.js的文件中写入，但是要在<script></script>标签中引入外联的.js文件；

        • 格式为：

          <script type="text/javascript" src="外联文件.js的路径" charset="UTF-8">
</script>

    • 常用语法：

      • var 变量名：创建一个变量；注意：Js中变量如果没有赋值，默认undefined。

      • 数据类型：

        • 基本类型：
          • Undefined：只有一个值undefined，当定义的变量未被赋值时，默认值就是undefined；
          • Null：只有一个值null，空值，但是在ECMAScript中，定义了undefined和null相等(从根本上还是不相等的)；
          • Boolean：有两个值true和false；
          • Number：表示任意数字，包括整数、小数等，都用Number表示；
          • String：字符串用””或’’声明；Js中没有字符类型；
        • 引用类型：
          • 是一种叫做类(class)的东西，但我们一般不会自己创建对象，而是直接使用，Js
            已经给我们准备好了，所以说Js是基于对象，而不是面向对象；

      • Js中用alert()方法直接弹出你想要输出的结果；

      • Js中的==和===的区别：

        • ==：只比较数值；

          例如：

          var a= "10";
var b= 10;
alert(a==b);

#弹出结果为true，因为a和b的的数值都是10，所以为true；

          ===：既比较数值，也比较类型，都相同才是全等。

      • Js中参数类型转为Boolean值：

        • Undefined：转为Boolean结果为false；
        • Null：转为Boolean结果为false；
        • Number：如果参数为+0、-0、NaN(即不是一个数字)，转为Boolean结果为false；否则为true；
        • String：如果参数为空字符串，则结果为false；否则为true；
        • Object：对象类型都为true。

      • Boolean类型参与运算：false是0，true是1；

      • Js中的函数(方法)的格式(注意：参数类型不需要写在方法中)：

        function 函数名(参数列表){
	函数体；
}

      • Js中没有方法重载，如果方法名相同，则后定义的方法会覆盖之前的方法；

      • 获取指定标签对象和内容：

        • var inputElement= document.getElementById()：该方法可以获取一个标签对象，然后通过该对象inputElement.value;获取(表单)元素标签的内容；
        • getElementsByName()：通过name属性值获得所有元素；
        • getElementsByClassName()：通过class属性值获得所有元素；
        • getElementsByTagName()：通过标签名获得所有元素。

      • 设置定时器：

        • window.setInterval(code,millisec)，其中：
          • code：可以是Js代码，也可以是一个函数；
          • millisec：时间间隔，单位是毫秒；即，每隔millisec毫秒会执行code一次。
        • window.setTimeout(code,millisec)，其中：
          • code：可以是Js代码，也可以是一个函数；
          • millisec：时间间隔，单位是毫秒；即，millisec毫秒后只执行code一次。

      • 页面加载事件：即window.onload事件，当页面加载完毕之后才会触发后面的函数；基本语法格式：

        #方式一:
window.onload = 函数名1;
function 函数名1(){
	函数体；
}

#方式二：
window.load = function(){
	函数体；
}

      • 修改标签的样式属性值：标签对象.style.属性 = 值；

      • 获取HTML中非表单元素标签的内容：

        • 标签对象.innerHTML：获取标签的所有内容，包括HTML标签，即会解析其中HTML代码；
        • 标签对象.innerText：获取标签的所有内容，不包括HTML标签，即不会解析其中HTML代码；

      • 事件：

        • onsubmit：提交按钮事件，提交按钮点击后，触发表单提交事件；
        • onfocus：获取焦点事件，即鼠标选中，就是焦点获取；
        • onblur：失去焦点时间，即鼠标未选中，就是焦点失去；
        • onmousevoer：鼠标移入事件，即鼠标移入时的事件；
        • onmouseout：鼠标移出事件，即鼠标移出的事件；

      • Js中的this关键字：表示的是当前元素，即谁调用，就代表谁；

      • Js中创建数组(这里的Array与Java中的集合类似，长度可变，可存任意类型)：new Array(元素1,元素2,元素3....)；

 

 

NO.1.4 JQuery

JQuery的存在极大地简化了JavaScript 编程。

  • JQuery

    • 定义：是一套跨浏览器的Js库；可以简化HTML和Js之间的操作；

    • 导入方式：使用外联式方法导入对应的“.js“后缀文件即可；

    • JQuery语法：

      • 基本语法格式：JQuery()或者$()；

      • 获取值：var v = $("选择器")，获取JQuery对象; v.val()，即可获取对应表单标签的值；

      • 页面加载事件：$(function(){函数体})；

      • 事件绑定：

        获取JQuery对象
JQuery对象.事件方法(function(){函数体})

    • JQuery效果：

      • 显示和隐藏：
        • show(毫秒值)；
        • hide(毫秒值)；
        • taggle(毫秒值)：这个是切换，即显示和隐藏相互切换；毫秒值是指，在指定毫秒值内，显示或隐藏；
      • 滑入和滑出：
        • slideDown(毫秒值)；
        • slideUp(毫秒值)；
        • slideToggle(毫秒值)：这个是切换，即滑入和滑出相互切换；毫秒值是指，在指定毫秒值内，滑入或滑出；
      • 淡入和淡出：
        • fadeIn(毫秒值)；
        • fadeOut(毫秒值)；
        • fadeToggle(毫秒值)： 这个是切换，即淡入和淡出相互切换；毫秒值是指，在指定毫秒值内，淡入或淡出；

    • JQuery选择器：

      • 基本选择器：
        • #id：选择指定的id的标签；
        • .class：选择指定的class值的标签；
        • 标签：选择指定的标签名的标签；
        • *：选择所有的标签；
        • a,b,c：选择指定a,b,c的标签；
      • 层次选择器：
        • a b：父标签a下所有的后代标签b；
        • a>b：父标签a下所有的子类标签b；
        • a+b：标签a后第一个b标签；
        • a~b：标签a后所有的b标签；
      • 过滤选择器：
        • a:first：第一个a标签；
        • a:last：最后一个a标签；
        • a:even：所有偶数a标签；
        • a:odd：所有奇数a标签；
        • a:eq(index)：a标签下第index个标签；
        • a:gt(index)：a标签下大于index的标签；
        • a:lt(index)：a标签下小于index的标签；
      • 内容过滤选择器：
        • a:has(内容)：a标签下所有含有”内容”的标签；
          • 属性选择器：
            • a:[属性=属性值]：a标签下属性=属性值的所有的标签；
          • 表单选择器：
            • :input：所有for表单下的子标签；

    • JQuery对CSS样式的操作：

      • JQuery对象.css("样式属性名")；
      • JQuery对象.css("样式属性名","样式属性值")。

    • JQuery对属性的操作：

      • JQuery对象.attr("属性名")；
      • JQuery对象.attr("属性名","属性值")；

    • JQuery遍历方式：

      JQuery对象.each(function(index,ele){  #index指的是要遍历的对象的索引值；ele指的是要遍历的每一个对象，也可以使用this指代每一个需要遍历的对象

	alert($(this).val());
})

    • 表单校验： 

      #在页面加载成功后
获取表单对象.validate({
	rules:{},  #校验规则
	messages:{}  #自定义提示信息
})

rules写法：
要校验的name属性：{
	校验器1：取值，
	校验器2：取值
}
注意：多个校验规则之间使用,分割

message写法：
要校验的name属性：{
	校验器1："自定义提示信息1"，
	校验器2："自定义提示信息2"
}
			
校验器：
required   true/false   必填校验
number     true/false   数字校验
min        数字          最小值
max        数字          最大值
range      数值区间       [最小值,最大值]
minLength|maxLength|rangLength   最小值|最大值|长度区间
email:  email           邮箱校验     *@*   1@x
equalTo:  jquery对象    重复性校验

    • 栅格系统：把每一行分为十二份 超过十二份自动换行；

      • 大屏幕：col-lg-n。如，col-lg-1，即1份为一格，大屏幕展示出来的就有12格；
      • 中等屏幕：col-md-n。如，col-md-3，即3份为一格，中等屏幕展示出来的就只有4格；
      • 小屏幕：col-sm-6。如，col-sm-6，即6份为一格，小屏幕展示出来的就只有2格；
      • 超小屏幕：col-xs-12。如，col-xs-12，即12份为一格，中等屏幕展示出来的就只有1格。
